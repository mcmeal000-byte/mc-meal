# MC Meal Mobile Notes

The build is responsive and includes mobile controls.

## Recommended mobile experience

- Landscape mode is recommended.
- Hub uses on-screen directional controls and ENTER button.
- Real mini-games keep their own mobile controls inside the game frame.
- The game can be played on mobile as a playable alpha preview.

## Later mobile optimization

For the production version, the next upgrade should be:
- Fullscreen mobile game mode.
- Separate route per mini-game.
- Better portrait fallback.
- Wallet connect mobile deep links.
